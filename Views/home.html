Welcome to My Website

I am excited to showcase key web development concepts, including HTML5 principles and techniques, as well as implementing routing with Express.js. This website serves as a demonstration of these fundamental web technologies in action.

Site Overview

This website consists of three static pages: Home, About, and HTML Demo. Additionally, it features five dynamic routes that provide structured data access:

  • All Students – Displays a list of all registered students.
  • Students by Course – Retrieves students enrolled in a specific course.
  • A Single Student by Student Number – Fetches details of a student based on their unique student ID.
  • Teaching Assistants – Lists all individuals in Teaching Assistants.
  • All Courses – Provides information about available courses.
